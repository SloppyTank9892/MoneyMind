# Quick API Key Setup

## Once You Have Your API Key

1. Open: `lib/config/api_keys.dart`

2. Find this line:
   ```dart
   static const String geminiApiKey = 'YOUR_GEMINI_API_KEY_HERE';
   ```

3. Replace it with:
   ```dart
   static const String geminiApiKey = 'PASTE_YOUR_KEY_HERE';
   ```

4. Save the file

5. Restart the app completely
